# Employee Dashboard

Internal HR and IT management dashboard built with React.

## Features

- Employee onboarding tracking
- IT asset management
- Fleet management interface
- HR analytics and reporting
- Integration with Microsoft 365

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Chart.js for analytics
- React Router
